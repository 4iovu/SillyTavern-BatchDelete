# SillyTavern-BatchDelete

SillyTavern 扩展插件，批量勾选并删除角色卡（支持同时删除关联世界书）及独立世界书。

---

## 功能

- **批量删除角色卡**：勾选任意数量的角色卡后一键删除
- **关联世界书处理**：删除角色卡前，弹窗列出该角色绑定的世界书，用户可自行勾选是否一并删除
- **批量删除世界书**：独立的世界书标签页，同样支持多选/全选删除
- **全选 / 取消全选**：工具栏一键全选
- **搜索过滤**：实时搜索角色卡或世界书名称
- **进度条**：删除过程中显示实时进度
- **安全确认弹窗**：防止误操作

---

## 安装

### 方式一：通过 SillyTavern 界面安装（推荐）

1. 打开 SillyTavern，点击顶栏右侧 **Extensions（扩展）** 图标（魔方图标）
2. 点击 **Install extension（安装扩展）**
3. 粘贴本仓库地址：
   ```
https://github.com/4iovu/SillyTavern-BatchDelete
```
4. 点击 **Save**，刷新页面

---

## 使用方法

1. 点击右侧面板的 **Extensions（扩展）** 图标展开扩展面板
2. 点击顶部的 **🗑 批量删除角色卡** 按钮
3. 在弹出的面板中：
   - 切换 **角色卡 / 世界书** 标签
   - 勾选要删除的项目（可全选）
   - 使用搜索框快速定位
   - 点击 **删除所选** 按钮
4. 如果选中的角色卡有绑定世界书，会弹出确认框供你选择是否一并删除
